🎮 控制理論

PID 控制

基礎

比例-積分-微分控制器,是無人機姿態控制的基礎算法。

u(t) = Kp·e(t) + Ki·∫e(t)dt + Kd·de(t)/dt
  • Kp (比例): 快速響應誤差
  • Ki (積分): 消除穩態誤差
  • Kd (微分): 預測未來誤差,減少超調
應用: 姿態穩定、高度控制、速度控制

LQR 控制

進階

線性二次調節器,通過最小化成本函數實現最優控制。

J = ∫(xTQx + uTRu)dt
  • Q 矩陣: 狀態權重
  • R 矩陣: 控制輸入權重
  • K 增益: 最優反饋增益矩陣
應用: 軌跡跟蹤、多變量控制、最優飛行

模型預測控制 (MPC)

進階

基於模型預測未來狀態,並優化控制序列的先進控制方法。

min Σ(||xk-xref||2 + ||uk||2)
  • 預測範圍: 未來 N 步預測
  • 約束處理: 可處理輸入/狀態約束
  • 滾動優化: 每步重新計算最優解
應用: 路徑規劃、避障、編隊飛行

🧭 導航與定位

卡爾曼濾波

中階

最優線性濾波器,融合多個感測器數據進行狀態估計。

x̂k = x̂k- + Kk(zk - Hx̂k-)
  • 預測步驟: 基於系統模型預測狀態
  • 更新步驟: 融合測量值修正預測
  • 卡爾曼增益: 平衡預測與測量
應用: GPS/IMU 融合、狀態估計、感測器融合

擴展卡爾曼濾波 (EKF)

進階

卡爾曼濾波的非線性擴展,通過線性化處理非線性系統。

Fk = ∂f/∂x|x̂k-1
  • 雅可比矩陣: 非線性函數線性化
  • 一階近似: 泰勒展開近似
  • 適用範圍: 弱非線性系統
應用: SLAM、視覺慣性里程計、姿態估計

粒子濾波

進階

基於蒙特卡羅方法的非線性非高斯濾波器。

p(xk|z1:k) ≈ Σwikδ(xk-xik)
  • 粒子表示: 用樣本近似分佈
  • 重要性採樣: 根據權重重採樣
  • 無線性假設: 適用任意非線性系統
應用: 全局定位、多目標追蹤、複雜環境導航

🗺️ 路徑規劃

A* 算法

中階

啟發式搜索算法,結合實際成本與估計成本尋找最優路徑。

f(n) = g(n) + h(n)
  • g(n): 起點到當前節點的實際成本
  • h(n): 當前節點到目標的啟發式估計
  • 最優性: h(n) 可採納時保證最優解
應用: 網格地圖路徑規劃、全局路徑搜索

RRT (快速探索隨機樹)

中階

基於採樣的路徑規劃算法,適用於高維空間和複雜約束。

xnew = xnear + ε · (xrand - xnear)/||xrand - xnear||
  • 隨機採樣: 在配置空間隨機採樣
  • 樹擴展: 向採樣點擴展樹結構
  • 概率完備: 存在解時必能找到
應用: 動態環境規劃、高維空間搜索、運動規劃

人工勢場法

基礎

將目標設為吸引力,障礙物設為排斥力,引導無人機運動。

F = Fatt + Frep
  • 吸引勢場: 目標點產生吸引力
  • 排斥勢場: 障礙物產生排斥力
  • 局部最小值: 可能陷入局部最優
應用: 實時避障、局部路徑規劃、編隊控制

👁️ 計算機視覺

視覺 SLAM

進階

同步定位與地圖構建,使用視覺感測器實現自主導航。

p(x0:t, m | z0:t, u0:t)
  • 前端: 特徵提取與匹配 (ORB-SLAM)
  • 後端: 位姿優化與迴環檢測
  • 地圖表示: 稀疏/稠密點雲地圖
應用: 室內導航、GPS 拒止環境、3D 重建

光流法

中階

通過分析連續影像中像素的運動估計相機或物體的運動。

I(x,y,t) = I(x+dx, y+dy, t+dt)
  • Lucas-Kanade: 稀疏光流法
  • Horn-Schunck: 稠密光流法
  • 亮度恆定假設: 像素亮度不變
應用: 視覺里程計、目標追蹤、避障

深度學習檢測

進階

使用卷積神經網絡進行目標檢測與識別。

y = σ(W * x + b)
  • YOLO: 實時單階段檢測器
  • Faster R-CNN: 兩階段高精度檢測
  • 語義分割: 像素級場景理解
應用: 目標識別、場景理解、自主降落

🤖 機器學習與優化

強化學習

進階

通過與環境互動學習最優策略的機器學習方法。

Q(s,a) ← Q(s,a) + α[r + γ max Q(s',a') - Q(s,a)]
  • Q-Learning: 值函數學習
  • Policy Gradient: 策略梯度方法
  • PPO/SAC: 現代深度強化學習
應用: 自主飛行、敏捷機動、任務規劃

群體智能優化

中階

模擬自然界群體行為的優化算法。

vi = w·vi + c1r1(pi-xi) + c2r2(g-xi)
  • PSO: 粒子群優化
  • ACO: 蟻群算法
  • GA: 遺傳算法
應用: 參數調優、路徑優化、任務分配

凸優化

進階

求解凸目標函數在凸集上的最優化問題。

minimize f(x) subject to gi(x) ≤ 0
  • 二次規劃: QP 問題求解
  • 半定規劃: SDP 優化
  • 內點法: 高效求解器
應用: 軌跡優化、控制器設計、資源分配
